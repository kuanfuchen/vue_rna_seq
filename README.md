# RNA-seq

## Project setup

```
Vue.js ( Vue 3 )

npm install core-js
npm install file-saver
npm install html2canvas
npm install jquery
npm install plotly.js
npm install plotly.js-dist
npm install plotly.js-dist-min
npm install rxjs
npm install vuetify
npm install sheetjs
npm install zoomist
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
